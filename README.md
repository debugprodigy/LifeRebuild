# LifeRebuild

**LifeRebuild** is a gamified self-improvement and habit tracking web app designed to help users rebuild their daily routine through habits, journaling, focus sessions, challenges, streaks, XP, badges, and progress reports.

It is built as a mobile-first Progressive Web App, so it can be hosted on GitHub Pages and installed on a phone like a native app.

---

## What is LifeRebuild?

LifeRebuild is a 90-day personal growth system wrapped inside a habit tracker.

Instead of simply checking off tasks, the app turns self-improvement into a game-like journey where users earn XP, level up, unlock badges, track streaks, complete daily challenges, write reflections, monitor focus time, and review their progress.

The goal is simple:

> Reset. Focus. Grow.

LifeRebuild is built for people who want to improve discipline, consistency, productivity, focus, health, reading, journaling, and overall life direction.

---

## Core Purpose

LifeRebuild helps users:

- Build better habits
